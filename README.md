Target Brazil E-Commerce Analysis
Project Overview
This project analyzes Target’s Brazil e-commerce dataset using Google BigQuery (SQL) to uncover trends in sales, customer distribution, logistics performance, freight cost, and payment behavior.
The objective was to derive data-driven operational and strategic insights to improve sales growth, logistics efficiency, and customer experience.
Business Objectives
•	Analyze order trends over time
•	Identify seasonal and daily purchasing patterns
•	Evaluate state-wise sales performance
•	Assess freight cost efficiency
•	Measure delivery performance vs estimated timelines
•	Understand customer payment behavior

Skills Used
•	SQL (Google BigQuery)
•	Data Aggregation & Window Functions
•	CTEs (Common Table Expressions)
•	Joins across multiple tables
•	Date & Time Functions
•	Case Statements for segmentation
Dataset Overview
Time Range: September 2016 – October 2018
Geographical Coverage: 27 Brazilian states | 4119 cities

Business Impact
This analysis provides actionable recommendations to:
•	Improve revenue growth strategy
•	Optimize logistics operations
•	Reduce freight costs
•	Enhance delivery efficiency
•	Strengthen customer retention
